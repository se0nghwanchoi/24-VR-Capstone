Built-In Render Pipeline Instrcutions:
Recommended Color Space: Linear (Project Settings -> Player -> Other Settings -> Color Space)
Recommended Rendering Path: Deferred
Import Post Processing from Package Manager. Assign Post Processing profiles from BuiltIn_RP folder. 

HDRP Instructions:
1. Import asset to an existing HDRP project.
2. Import HouseProps2_HDRP.unitypackage file from Assets -> Import Package -> Custom Package.
3. Replace all files.
4. Wait for shaders and scene to update to HDRP.

URP Instructions:
1. Import asset to an existing URP project.
2. Import HouseProps2_URP.unitypackage file from Assets -> Import Package -> Custom Package.
3. Replace all files.
4. Wait for shaders and scene to update to URP.
